# Keralux — ონლაინ კატალოგი

Keralux-ის თანამედროვე, მობილურზე მორგებული ონლაინ კატალოგისა და შეკვეთის დემო ვერსია.

## შესაძლებლობები

- პროდუქციის კატეგორიები და ფილტრაცია
- ძიება დასახელებით, კოდით, ბრენდით, მასალითა და საკვანძო სიტყვებით
- რჩეულები და კალათა `localStorage`-ში შენახვით
- პროდუქტის სწრაფი ნახვა და ტექნიკური მახასიათებლები
- WhatsApp-ით შეკვეთისა და პროექტის კონსულტაციის მოთხოვნა
- მისამართი, ტელეფონი, Facebook და Google Maps ბმულები
- მობილური, პლანშეტისა და კომპიუტერის ადაპტური დიზაინი
- SEO მეტამონაცემები, LocalBusiness schema, manifest და service worker
- GitHub Pages-ის ავტომატური განთავსების workflow

## ადგილობრივად გაშვება

```bash
python -m http.server 8080
```

შემდეგ გახსენით `http://localhost:8080`.

## GitHub Pages

რეპოზიტორის **Settings → Pages** განყოფილებაში Source უნდა იყოს **GitHub Actions**. `main` ბრენჩზე ყოველი ცვლილების შემდეგ workflow ავტომატურად განათავსებს საიტს.

## შენიშვნა

ეს არის საჩვენებელი ვერსია. საბოლოო გაშვებამდე საჭიროა პროდუქციის რეალური კატალოგის, ზუსტი ფასების, მარაგის, მიწოდების წესებისა და გადახდის სისტემის შეთანხმება.
